СИМУЛЯТОР ПАШИ ЕГОРОВА
Задача: реализация игры(аля ферма) симулятора Паши Егорова. Суть - создание курсов для ФИИТа и их поддержка. 

Техническое задание: 
1. Изначально - пустой экран с ячейками для курсов. Часть из них недоступна. 
2. При нажатии на пустую ячейку открывается меню с созданием курса. Там можно выбрать кастомное название, преподавателя(с характеристиками), сложность курса, часы в неделю(реальные затраты на прохождение курса)
3. При нажатии на готовый курс открывается окно поддержки - для поддержки существующего курса(условно заменить препода, удалить курс,уменьшить сложность и тп)
4. В игре есть “валюта” - КонтурКоин. За них можно будет открывать новые ячейки для курсов, звать на работу преподов, редачить курсы и тп)
5. У курсов есть рейтинг, его можно повышать, покупая улучшения. Также курсы в зависимости от рейтинга приносят определенное количество КонтурКоинов. 
6. Для победы в игре необходимо закрыть все ячейки так, чтобы у всех созданных курсов был рейтинг больше 75%(цифра может поменяться)
7. * Звонок в деканат для решения вопроса с физкультурой
8. * Провести опросик для какого-либо курса с целью улучшения рейтинга
9. * Создать инсту с Егором Макаровым для прихода КонтурКоинов
10. * Создать возможность зарабатывать КонтурКоины с помощью игры жизнь
